1. Crear un formulario con:
* Nombre (obligatorio)
* Mensaje (obligatorio, máximo 200 caracteres)
* Email (opcional, pero válido si lo ingresa)
2. Dar estilo particular al formulario
3. Validar campos
4. Mostrar lista con errores creando elementos del DOM
5. Cambiar estilo a campos con errores
6. Al enviar, si los datos está bien, agregar mensaje a una lista de mensajes, con el siguiente formato:
Si proveyó el email,
“nombre dice mensaje. Contacto: email.” Si no, “nombre dice mensaje.”
7. Limpiar los campos, errores de validación y cambios de estilo
8. Hacer foco en el primer input al iniciar el documento y luego de cada envío correcto
NOTA: en el formulario, conviene usar el evento "onsubmit" para llamar a la función que hace la validación. A los efectos del práctico, la función de validación siempre debe devolver false, así el formulario no se envía. En el onsubmit, hacer de esta manera:
form onsubmit="return validar()"
Es decir, haciendo return de la función para validar que haya definido.
